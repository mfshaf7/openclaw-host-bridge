# Records

This directory stores historical evidence for security-significant or
production-impacting source changes in `openclaw-host-bridge`.

Use change records when a bridge-layer change affects:

- host trust-boundary enforcement
- allowed-root or permission-class behavior
- audit logging or runtime attestation
- stage or prod bridge launch and operator recovery workflow

For the structured record format, see
[change-records/README.md](change-records/README.md).
